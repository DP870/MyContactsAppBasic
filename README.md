#### UC-01:
This use case implements user registration for the My Contacts App.
#### UC-02: User Authentication
Actor: Registered User
Description: User logs in with credentials to access their contact list.
#### UC-03: User Profile Management
Actor: Logged-in User
Description: User updates profile information.
#### UC-04: Create Contact
Actor: Logged-in User
Description: User adds a new contact with name, phone numbers, email addresses.
#### UC-05: View Contact Details
Actor: Logged-in User
Description: User views complete information of a specific contact.
#### UC-06: Edit Contact
Actor: Logged-in User
Description: User modifies existing contact information.
#### UC-07: Delete Contact
Actor: Logged-in User
Description: User removes a contact from their list with confirmation.
#### UC-08: Bulk Operations
Actor: Logged-in User
Description:
User performs operations on multiple contacts.
##### UC-09: Search Contacts
Actor: Logged-in User
Description:
User searches contacts by name.
#### UC-10: Basic Filtering
Actor: Logged-in User
Description:
User applies filters.
#### UC-11: Create and Manage Tags
Actor: Logged-in User
Description:
User creates custom tags (Family, Work, Friends) for organizing contacts.
##### UC-12: Apply Tags to Contacts
Actor: Logged-in User'
Description:
User assigns one or multiple tags to contacts.
